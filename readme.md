This is a simple Snake game built using HTML, CSS, and JavaScript as a learning project.

The game is still a work in progress and is not fully completed yet.

## Features
- Basic snake movement using keyboard controls (W, A, S, D)
- Food spawning at random positions
- Simple game loop using `requestAnimationFrame`

## How to Run
1. Clone or download this repository.
2. Open the `index.html` file in your browser.
3. Use **W, A, S, D** keys to move the snake.

## Purpose
This project was created to practice JavaScript and understand basic game logic.
